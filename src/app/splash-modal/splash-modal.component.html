Welcome to New EPIC!

We have recently updated our Project Information Centre and are actively releasing new features to help you find project information. In the interim, if you cannot find the documents you are looking for please contact us at epicsupport.eao@gov.bc.ca.

To find a project:

  • Click on “Find Environmental Assessment Projects”, Enter your project name in the search field on the mapping interface and select the map pin
  • A pop up with give you some details regarding the project selected, click “View Project Details” to go to the project landing page
  • Or you can search by “List of Projects” and enter your project name in the search field here.

Project Landing Pages:

Project information is now split over 3 tabs

  • Project Information
  • Commenting
  • Documents

Documents are now searchable, there is a search help located in the header to assist you.
